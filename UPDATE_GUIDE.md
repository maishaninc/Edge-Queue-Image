## 🎯 布局算法

每组12张图片的布局有4种配置，循环使用：

**配置1**: 大图左上 + 长图右上
```
[大][大][1][2]
[大][大][长][长]
[3][4][5][6]
```

**配置2**: 大图右上 + 长图左中
```
[1][2][大][大]
[长][长][大][大]
[3][4][5][6]
```

**配置3**: 大图左中 + 长图右下
```
[1][2][3][4]
[大][大][5][6]
[大][大][长][长]
```

**配置4**: 大图右下 + 长图左上
```
[长][长][1][2]
[3][4][5][6]
[7][8][大][大]
```
